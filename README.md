# Sistema de Pedidos – JavaFX

Este projeto é um sistema simples de pedidos feito em Java e JavaFX.  
O usuário pode escolher o filme, selecionar os assentos e confirmar o pedido.  
Também existe uma tela para visualizar todos os pedidos registrados.

## Como executar

1. Instale Java 17 ou superior.
2. Instale o JavaFX 17 ou superior.
3. Abra o projeto na sua IDE (IntelliJ, Eclipse, NetBeans, VScode, etc.).
4. Execute a classe APP()

## Testes

Para testar:
1. Abra o programa.
2. Faça um pedido completo.
4. Vá na tela de pedidos e verifique se o pedido aparece na lista.
